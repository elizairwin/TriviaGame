# Trivia-Game

## Description: 
A responsive trivia game that has a food theme. The user begins the game and has 30 seconds to answer each question. At the end, a scoreboard is returned for the user.

## Technology Used:
* HTML
* CSS
* Bootstrap
* JavaScript
* jQuery
* Timers
